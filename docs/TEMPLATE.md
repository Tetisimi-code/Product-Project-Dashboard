# Word Template Placeholder

Use the existing organization Word template with a single content insertion marker.

## Default placeholder

Place this exact placeholder on a blank line where the generated documentation
should be inserted:

`{{DOCUMENT_CONTENT}}`

## Template file

Place the organization template at:

- `docs/template.docx`

## Notes

- Do not change branding, headers, footers, or styles in the template.
- The assembler will replace the placeholder with rendered module content.
- Keep the placeholder unique and used only once in the template.
