## ‚ùå **Can't Login - "Invalid Credentials"**

### **Issue: Password not working**

**Symptoms:**
- "Invalid email or password" error
- Can't access account
- Sure you have the right password

### **Solutions:**

**Solution 1: Use Forgot Password**
```
1. Click "Forgot Password?" on login page
2. Enter your email address
3. Check email (including spam folder)
4. Click reset link
5. Set new password
6. Try logging in
```

**Solution 2: Double-Check Email**
```
- Verify email address spelling
- Check for typos
- Try variations (with/without dots)
- Ensure correct domain
- Gmail ignores dots in address
```

**Solution 3: Try Different Browser**
```
- Clear browser cache
- Try incognito/private mode
- Use different browser
- Test on different device
```

**Solution 4: Contact Admin**
```
- Admin can verify your account exists
- Admin can reset your account
- Admin can recreate account if needed
- Provide your exact email address
```

4. Complete verification

**Solution 3: Check With Admin**

2. Provide email address
3. Admin checks user database
4. Admin can verify if account exists
5. Admin can recreate if deleted

## üîí **Password Reset Not Working**

### **Issue: Can't reset password**

See **[Email Issues - Password Reset](./EMAIL_ISSUES.md#-password-reset-email-not-received)** for detailed email troubleshooting.

**Quick Solutions:**

**If Email Not Received:**
```
1. Check spam/junk folder
2. Wait 10-15 minutes
3. Request reset again
4. Try different email address
5. Contact admin for help
```

**If Reset Link Expired:**
```
1. Links expire after 1 hour
2. Request new reset link
3. Use link immediately
4. Don't click old links
```

**If Reset Link Not Working:**
```
1. Copy full URL (don't click)
2. Paste in new browser tab
3. Try incognito mode
4. Clear browser cache
5. Try different browser
```

## ‚è±Ô∏è **Session Expired**

### **Issue: Logged out automatically**

**Symptoms:**
- "Session expired" message
- Redirected to login
- Lost your work
- Need to login again

**This is Normal!**
- Sessions expire for security
- Typically after 24 hours of inactivity
- Or after browser close (depending on settings)

### **Solutions:**

**Solution 1: Just Login Again**
```
- Enter your email and password
- Your data is saved in cloud
- You won't lose anything
- Pick up where you left off
```

**Solution 2: Use "Remember Me"** (if available)
```
- Check "Remember Me" box when logging in
- Stays logged in longer
- Still expires eventually for security
```

**Solution 3: Keep Tab Open**
```
- Don't close browser tab
- Minimizes session expiration
- Still may expire after long time
- Just login again if needed
```

## üîê **Account Locked**

### **Issue: Too many failed login attempts**

**Symptoms:**
- "Account temporarily locked"
- "Too many attempts"
- Can't login even with correct password

**This is a Security Feature!**
- Protects against brute force attacks
- Temporary lockout (usually 15-30 minutes)

### **Solutions:**

**Solution 1: Wait and Try Again**
```
1. Wait 15-30 minutes
2. Don't keep trying (extends lockout)
3. Use correct password
4. Should work after waiting
```

**Solution 2: Use Password Reset**
```
1. Click "Forgot Password?"
2. Reset your password
3. May bypass lockout
4. Set new secure password
```

**Solution 3: Contact Admin**
```
1. Admin may be able to unlock
2. Provide your email address
3. Admin verifies your identity
4. May need to reset password
```

**Solution 3: Contact Admin**

3. Admin may need to grant access

## üîÑ **Stuck on Loading Screen**

### **Issue: Login page loads but spins forever**

**Symptoms:**
- Loading spinner doesn't stop
- Page never finishes loading
- Can't interact with page
- Blank screen with spinner

### **Solutions:**

**Solution 1: Hard Refresh**
```
- Windows: Ctrl + Shift + R
- Mac: Cmd + Shift + R
- Clears cached version
- Forces fresh load
```

**Solution 2: Clear Browser Cache**
```
1. Chrome: Settings ‚Üí Privacy ‚Üí Clear browsing data
2. Select "Cached images and files"
3. Clear last 24 hours
4. Try login again
```

**Solution 3: Check Browser Console**
```
1. Press F12
2. Go to Console tab
3. Look for errors (red text)
4. Share errors with admin
```

**Solution 4: Try Different Browser**
```
- Chrome
- Firefox
- Safari
- Edge
- Use incognito mode
```
